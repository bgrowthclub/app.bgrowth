# BGrowth Homepage

React + Vite + TypeScript + Tailwind + Framer Motion + Lucide icons.

## Run locally
```
npm install
npm run dev
```

## Build for production
```
npm run build
```

## Deploy
Push this folder to GitHub and connect it to Vercel (same flow as BGrowth Studio) — no extra config needed, Vercel auto-detects Vite.
